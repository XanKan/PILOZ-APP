# Release 0.9.0-compliance.5

Date : 23 juillet 2026
Schéma : `202607230051`

Cette version ajoute les espaces de gestion client et catalogue, sans modifier rétroactivement les documents finalisés. Les unités disponibles dans les articles et documents sont configurables par entreprise et sélectionnées dans une liste fermée. Les contacts et adresses choisis restent conservés dans le snapshot du document. Les profils en lecture seule restent limités à la consultation.

Cette release reste une pré-version. Elle ne constitue ni une certification NF 525/NF 203, ni une validation juridique ou comptable.
